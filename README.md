# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior using transactional data (~3,900 records) to understand purchasing patterns, customer segments, and subscription behavior.

The goal is to generate insights that support better business decisions and data-driven strategies.

---

## Tools Used

* Python (Pandas)
* MySQL
* Power BI
* Gamma (for presentation & reporting)

---

## Data Preparation

* Cleaned and explored data using Python
* Handled missing values in review ratings
* Created features such as age groups and customer segments
* Loaded cleaned data into MySQL for analysis

---

## Analysis (SQL)

Performed analysis in MySQL to answer key business questions:

* Revenue by gender
* Top products by rating
* Discount usage across products
* Customer segmentation (New, Returning, Loyal)
* Subscription vs non-subscription behavior
* Revenue by age group

---

## Dashboard & Reporting

* Built an interactive dashboard in Power BI to visualize:

  * Number of customers
  * Average purchase amount
  * Average review rating
  * Revenue and sales by category
  * Subscription distribution
  * Revenue and sales by age group

* Created a presentation in Gamma to communicate insights and business recommendations clearly

---

## Key Insights

* Clothing category generates the highest revenue
* Most customers are not subscribed
* Young adults contribute the highest revenue
* Some products rely heavily on discounts

---

## Business Recommendations

* Improve subscription offers
* Focus on loyal customers
* Optimize discount strategies
* Target high-value customer segments
